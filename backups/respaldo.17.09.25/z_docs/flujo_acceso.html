🤖 3. Lo que Hace el Sistema Automáticamente

📧 Cuando usuario solicita código:

1 Verificar email autorizado
if email in authorized_emails: # Continuar proceso
2 Generar código aleatorio
code = generate_access_code() # Ej: "ABC123"
3 Guardar código temporalmente
auth_codes.json: { "cliente@empresa.com": { "code": "ABC123", "timestamp": 1694257200, "used": false } }
4 Enviar email automáticamente
El sistema usa tu Gmail para enviar el código al usuario.
Email enviado contiene:
• Código de acceso (ABC123)
• Tiempo de expiración (10 minutos)
• Instrucciones de uso
• Información de contacto

🔐 Cuando usuario introduce código:

1 Verificar código válido
if codes[email]['code'] == 'ABC123' and not expired: # Código correcto y no expirado
2 Marcar código como usado
codes[email]['used'] = True # Evitar reutilización
3 Autorizar usuario y crear sesión
st.session_state.authenticated = True st.session_state.user_email = email st.session_state.auth_timestamp = time.time()
